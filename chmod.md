File permission
===

file owner
---
- User
- Group
- Everyone

permission
---
- r - read permission
- w - write permission
- x - execute permission

permission number
---
|number|permission |
|------|-----------|
|4     |r(ead)     |
|2     |w(rite)    |
|1     |(e)x(ecute)|


chmod
---
basic usage
```
chmod NNN [file]
```
